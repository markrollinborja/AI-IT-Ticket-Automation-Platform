# Project Roadmap

## Overview

The AI IT Ticket Automation Platform is being developed as a portfolio project that demonstrates enterprise backend engineering, workflow automation, API integrations, and practical AI-assisted business process automation.

The roadmap is intentionally divided into versions to maintain a focused scope while maximizing engineering quality and hiring value.

---

# Version 1 (MVP)

## Goal

Deliver a production-ready workflow automation platform that integrates with Jira Cloud and uses a hybrid Rule Engine + AI architecture for ticket priority classification.

### Core Platform

- ✅ FastAPI backend
- ✅ PostgreSQL database
- ✅ SQLAlchemy ORM
- ✅ Alembic migrations
- ✅ Docker Compose
- ✅ Environment configuration

### Jira Integration

- ✅ Jira Cloud webhook integration
- ✅ Jira-first architecture
- ✅ Automatic Jira priority updates

### Workflow Engine

- ✅ Workflow orchestration
- ✅ Ticket persistence
- ✅ WorkflowRun tracking
- ✅ Audit logging

### Rule Engine

- ✅ Deterministic priority classification

### Artificial Intelligence

- ✅ OpenAI integration
- ✅ GPT-4o-mini
- ✅ AI fallback classification
- ✅ Priority classification only
- ✅ AI confidence score persistence

### Notifications

- ✅ Slack notifications

### Monitoring

- ✅ Dashboard
- ✅ Workflow details page
- ✅ Timeline
- ✅ Workflow status tracking
